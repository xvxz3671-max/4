# Инструкции по установке и запуску

## 1. Установка Node.js

Скачайте и установите Node.js с официального сайта:
https://nodejs.org/

Рекомендуется версия LTS (Long Term Support).

После установки проверьте:
```bash
node --version
npm --version
```

## 2. Установка зависимостей

В корневой директории проекта выполните:
```bash
npm install
```

## 3. Настройка переменных окружения

Скопируйте файл с примерами:
```bash
copy .env.example .env
```

Отредактируйте `.env` файл:
- Получите BOT_TOKEN от @BotFather в Telegram
- Остальные переменные можно оставить по умолчанию для локальной разработки

## 4. Инициализация базы данных

```bash
npm run db:generate
npm run db:push
```

## 5. Запуск проекта

```bash
npm run dev
```

Это запустит:
- Web app: http://localhost:3000
- API: http://localhost:3001  
- Bot: в режиме polling

## 6. Тестирование

Откройте http://localhost:3000 в браузере для тестирования Mini App.

Для полного тестирования в Telegram:
1. Создайте бота через @BotFather
2. Получите токен и добавьте в .env
3. Запустите бота командой /start

## Структура проекта готова!

Все файлы созданы и настроены для production-ready деплоя на Vercel.
